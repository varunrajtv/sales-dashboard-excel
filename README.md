# Sales Dashboard in Excel

## Overview

This project is a Excel dashboard created to analyze sales performance using pivot tables, charts, slicers, and KPI cards. It focuses on transforming raw sales data into clear business insights through interactive visualizations.

## Features

* Sales performance analysis dashboard
* KPI cards for key business metrics
* Pivot Tables and Pivot Charts
* Interactive slicers for filtering
* Clean dashboard layout and business reporting

## Dashboard Components

* Total Sales
* Total Revenue / Profit 
* Quantity Sold
* Monthly Sales Trend
* Sales by Category / Product / Region 
* Interactive slicers for filtering

## Excel Skills Used

* Data Cleaning
* Pivot Tables
* Pivot Charts
* Slicers
* KPI Cards
* Dashboard Design
* Basic Excel Formulas

## Outcome

Built a interactive sales dashboard in Microsoft Excel to practice business reporting, data visualization, and dashboard design for data analyst portfolio development.
